    Second Take

    My friends and I often have heated debates with topics in sports (the GOAT in the NBA, whether Tom Brady is a system QB, etc.) and they usually are in-depth and fun to participate in. In this project, I wanted to replicate those same discussions online where several people can participate and enjoy it in a social media form.

    ⦾ Using React as a front-end library, Redux for state management (as well as creating and building UI), and Firebase/Express for backend services. Material-UI for icons, styling, etc.

    ⦾ Has functionalities of a modern-day social media platform: users can post their takes (a term for view-points on sports topics), like other takes (and unlike), and comment on takes. The user has the ability to add and change their profile picture, add a short bio, list their favorite team, and receive real-time notifications for when they receive engagement on their takes.

    ⦾ User authentication is done on Firebase and uses standards such as OAuth 2.0 and OpenID Connect to securely save user data in the cloud.

    ⦾ Login and signup pages also ensure that passwords match, user handles are not reused, no fields are left blank, and inputted information is correct.
    React.JS Firebase Redux.JS Express.JS

  • Backtrading Stocks
    Algorithm

    Backtesting Stocks Algorithm

    This project used backtesting algorithms to compare the strategy of buying stocks on a dip to see whether this strategy holds for a certain stock over time.

    ⦾ Backtesting is a way to test a strategy by seeing how it would fare using historical data. This way we can be more confident in employing the strategy in the stock market.(https://www.investopedia.com/terms/b/backtesting.asp)

    ⦾ In this project, I utilized the backtrader framework in Python which gave me the ability to run a strategy against a data feed to visualize our outcome

    ⦾ The strategy I used was relatively simple: buying the stock after a dip (prices fall for 2 consecutive days) and selling it after 5 days.

    ⦾ To test this with stocks, you can find historical data on any stock in Yahoo finance and download a CSV file for testing

    ⦾ We can input any stock and our portfolio value and see how the strategy would fair using backtesting by setting a time frame to test.
    Python | Trading Strategy Data Science
